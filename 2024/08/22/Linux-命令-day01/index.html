01 操作系统

1.1 介绍

说明：所有硬件设备组装完成后的第一层软件，能够使用户使用硬件设备的软件即为操作系统

1.2 分类

桌面操作系统：Windows/macOS/Linux
移动端操作系统: Android (安卓)/iOS (苹果)
服务器操作系统：Linux/Windows Server
嵌入式操作系统：Android (底层是 Linux)

02 Linux 系统

说明：在服务器端最常见的操作系统

2.1 Linux 发行版本

说明：在原版 Linux 系统的基础上，额外增加一些常用软件的操作系统
注释：类似于原生安卓系统和现行的华为系统/小米系统等之间的关系

2.2 文件和路径

说明：不同于 Windows 系统的⽂件路径的起点是盘符，Linux 系统的路径起点是’/'根目录
注意：

  1. / 为根目录，是所有文件的路径的起点
  2. 根目录下的所有第一层级的文件夹，均不能随意进行修改
  3. 掌握常用目录的作用即可
  4. / 根目录，/home 普通用户所在文件夹，/root 系统管理员所在文件夹

03 远程连接操作

说明：通过[远程连接⼯具]连接到服务器的操作即为远程连接
条件：
1> 服务器的 IP 地址
2> 服务器的账号和密码
注意：必须确保有网络连接条件

3.1 常见的远程连接工具

1 > FinalShell（推荐）：功能强大，有颜值，好操作 (高级功能需要订阅)
2 > XShell：个人用免费、商用收费
3 > Putty：免费开源的
4 > SecureCRT：收费
5 > MobaXterm：免费

3.2 远程连接基本步骤

入职后需要向相关人员 (运维/开发) 获取服务器 IP 地址及账号密码等信息
配合远程连接工具 (根据公司要求使用对应工具)，配置连接使用服务器即可

3.3 远程连接操作步骤

获取服务器 IP 地址
打开服务器终端输入指令 ifconfig 获取 IP 地址信息
默认可使用 ip addr 获取 IP 地址
在本地测试连通性
在本机使用 ping 命令行来测试与服务器的连通性

3.4 远程连接工具设置步骤

新建 --> SSH 连接
远程连接依赖的命令 SSH

04 Linux

4.1 Linux 命令学习原则

不要死记硬背 Linux 命令
务必要明确命令的基本组成形式

4.2 Linux 命令操作技巧

善用 tab 键进行文件名 / 路径名 / 命令的自动补全
使用 ↑ ↓ 快速获取先前输入过的命令
命令开始执行后，若需要取消执行：Ctrl + C

4.3 Linux 命令的基本组成

# 命令 [-选项] [参数]
command [-options] [parameter]
说明：
command : 命令主体
[-options] : 命令选项(可选), 根据命令可以显示更更加丰富的数据
[parameter] : 命令参数(可选)，命令的操作对象，一般文件、目录、用户和进程等都可以作为参数被命令操作

05 Linux 命令 - 案例 1

ls/pwd/cd/touch/mkdir/cp/mv/rm

Part 1 ls/pwd/cd/touch/mkdir

pwd – 查看当前路径位置
ls – 查看当前目录下有哪些文件和文件夹
mkdir – 创建一个文件夹/创建多个文件夹
cd – 切换到 xx 目录下
touch – 创建一个文件

Part 2 cd 其他用法和 mv

cd – 切换目录 --> 相对路径/绝对路径
mv – 默认用于移动文件到某一路径下，但如果目标文件名不是文件夹名或者文件名不存在，则视为修改文件名操作 --> mv file aa （将 file 更名为 aa）

相对路径和绝对路径

相对路径：凡是路径信息以 . 或者 .. 开头的均为相对路径
绝对路径：凡是路径信息以 / 或者 ~ 开头的均为绝对路径
注释：
.：当前路径下，一般省略
..：上一层目录
/：根目录
~：当前用户的家目录

Part 3 cp复制文件和文件夹

复制文件：cp aa adir 复制文件 aa 到 adir 目录下
复制文件夹：cp -r bdir/ cdir/ 复制文件夹 bdir 到 cdir 下

Part 4 mv/touch 创建多个文件/rm 删除文件和文件夹及所有文件

mv aa cdir/bdir/ – 移动文件 aa 到 bdir 文件夹
touch bb cc – 创建多个文件 bb cc
rm -i bb – 删除 bb 文件
rm -ir adir/ – 删除 adir 文件夹
rm -f aa – 强制删除
