![](E:\HazelDocuments\Blog\source_posts\img\Linux-命令-day03\Linux 命令 – day03_image_4.png)

修改文件权限：字母法

组别: u 拥有者 g 用户组 o 其他⼈ a 以上全部
修改: + 增加权限 - 删除权限 = 赋予权限
权限: r 可读 w 可写 x 可执行 - ⽆权限
![](E:\HazelDocuments\Blog\source_posts\img\Linux-命令-day03\Linux 命令 – day03_image_5.png)

  1. 必须明确文件权限的含义
  2. 使用数字法或者字母法均可
  3. 尽量不要随意赋予文件最高权限：数字法 777 字母法 rwxrwxrwx
  4. 建议适当修改文件权限：数字法 755 字母法 rwxr-xr-x，除拥有者外均无可写权限，以此来保证文件内容安全

yum 软件包管理工具

==yum==（ Yellow dog Updater, Modified）：是⼀个在 Linux 系统中常用的软件包管理器。
yum 提供了查找、安装、删除某⼀个、⼀组甚⾄全部软件包的命令，⽽且命令简洁又好记。
常用命令：
yum list: 列出所有可安装的软件清单
yum search: 查找软件包
yum install : 安装指定的软件
yum update : 更新指定的软件
yum update: 更新所有软件
yum remove : 删除软件包

数据库

说明: 数据库是专门用来存储数据的软件
注意: 对于测试工作而言, 如果项目页面没有实现, 但是我们又想要校验数据,则可以直接通过查询数据库实现
关系: 具体存在的商品录⼊后 -> 产⽣对应的数据(存到数据库中) -> 最后会被加载到项目页面中

数据库分类

分类:
1> 关系型数据库: 以数据表为核心
2> ⾮关系型数据库: 不存在数据表的概念
关系型数据库: RDMS(Relational Database Management System)关系型数据库系统
常见的关系型数据库:
Oracle: 在⼤型项⽬中使⽤，例如：银⾏、电信等项⽬
MySQL: Web 项⽬中使⽤最⼴泛的关系型数据库
Microsoft SQL Server: 在微软的项⽬中使⽤
SQLite: 轻量级数据库，主要应⽤在移动平台
关系型数据库的核心要素:
数据行（⼀条记录）
数据列（字段）
数据表（数据行的集合）
数据库（数据表的集合，⼀个数据库中能够有 n 多个数据表）

Linux 命令 6: netstat/lsof

==端口号==：
说明：想要连接至计算机 – 需要使用 IP 地址；想获取在计算机上运行的程序，需要通过端口号
注意：同⼀台计算机不能有多个程序同时使用同一个端口号，因此在运行程序时，如果提示“端口号被占用”，需要先找到占用端口号的程序并结束掉，然后才能运行目标程序
常见端口号：
1> HTTP: 80
2> HTTPS: 443
3> SSH: 22
4> MySQL : 3306

05 Linux 命令 - 案例 1

ls/pwd/cd/touch/mkdir/cp/mv/rm

Part 1 ls/pwd/cd/touch/mkdir

pwd – 查看当前路径位置
ls – 查看当前目录下有哪些文件和文件夹
mkdir – 创建一个文件夹/创建多个文件夹
cd – 切换到 xx 目录下
touch – 创建一个文件

Part 2 cd 其他用法和 mv

cd – 切换目录 –> 相对路径/绝对路径
mv – 默认用于移动文件到某一路径下，但如果目标文件名不是文件夹名或者文件名不存在，则视为修改文件名操作 –> mv file aa （将 file 更名为 aa）

相对路径和绝对路径

相对路径：凡是路径信息以 . 或者 .. 开头的均为相对路径
绝对路径：凡是路径信息以 / 或者 ~ 开头的均为绝对路径
注释：
.：当前路径下，一般省略
..：上一层目录
/：根目录
~：当前用户的家目录

Part 3 cp复制文件和文件夹

复制文件：cp aa adir 复制文件 aa 到 adir 目录下
复制文件夹：cp -r bdir/ cdir/ 复制文件夹 bdir 到 cdir 下

Part 4 mv/touch 创建多个文件/rm 删除文件和文件夹及所有文件

mv aa cdir/bdir/ – 移动文件 aa 到 bdir 文件夹
touch bb cc – 创建多个文件 bb cc
rm -i bb – 删除 bb 文件
rm -ir adir/ – 删除 adir 文件夹
rm -f aa – 强制删除
